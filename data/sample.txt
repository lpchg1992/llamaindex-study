# LlamaIndex 简介

LlamaIndex（原名 GPT Index）是一个数据框架，用于构建大型语言模型（LLM）应用。它提供了一套工具，使开发者能够轻松地将私有数据与 LLMs 结合使用。

## 核心概念

### 1. 文档加载（Document Loading）
LlamaIndex 支持从多种来源加载文档：
- 本地文件（txt, pdf, docx, md 等）
- 网络内容（网页、API）
- 数据库（SQL, NoSQL）
- 云存储

### 2. 索引构建（Indexing）
LlamaIndex 提供多种索引类型：
- **向量索引 (Vector Index)**：基于语义相似度检索
- **摘要索引 (Summary Index)**：用于快速概览
- **树形索引 (Tree Index)**：层次化组织文档
- **关键词索引 (Keyword Index)**：基于关键词匹配

### 3. 查询引擎（Query Engine）
查询引擎将索引与 LLM 结合，提供问答能力：
- 支持单步查询和多步查询
- 支持流式输出
- 支持自定义提示词

## 使用场景

1. **问答系统 (Q&A)**：基于私有文档回答问题
2. **聊天机器人**：结合文档的对话系统
3. **文档摘要**：自动生成文档摘要
4. **信息检索**：语义化的搜索引擎

## 快速开始代码示例

```python
from llama_index.core import VectorStoreIndex, SimpleDirectoryReader

# 加载文档
documents = SimpleDirectoryReader("./data").load_data()

# 构建索引
index = VectorStoreIndex.from_documents(documents)

# 创建查询引擎
query_engine = index.as_query_engine()

# 查询
response = query_engine.query("你的问题")
print(response)
```
